# Etch-a-Sketch

A fun, interactive drawing application inspired by the classic Etch A Sketch toy. Create colorful pixel art with random RGB colors and progressive opacity effects, all built with vanilla HTML, CSS, and JavaScript.

## Features

- **16x16 pixel drawing grid** - 256 cells for your creativity
- **Rainbow drawing mode** - Each stroke uses a random RGB color
- **Progressive opacity** - Colors intensify the more you draw over a cell
- **Clear button** - Reset the canvas instantly
- **No dependencies** - Pure HTML, CSS, and JavaScript

## Demo

Open `index.html` in your browser and start drawing by moving your mouse over the grid!

## Getting Started

### Prerequisites

- Any modern web browser (Chrome, Firefox, Safari, Edge)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/etch-a-sketch.git
   cd etch-a-sketch
   ```

2. Open `index.html` in your browser

No build process, server, or npm installation required.

## How to Use

1. **Draw** - Hover your mouse over the grid cells to paint with random colors
2. **Layer colors** - Go over the same cell multiple times to increase opacity
3. **Clear** - Click the "Clear" button to reset the canvas to white

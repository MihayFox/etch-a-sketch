# Etch-a-Sketch

A digital recreation of the classic Etch A Sketch toy, built with vanilla HTML, CSS, and JavaScript.

## Features

- 16x16 pixel drawing grid
- Draw by hovering your mouse over cells
- Clear button to reset the canvas
- No dependencies required

## Getting Started

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd etch-a-sketch
   ```

2. Open `index.html` in your browser

That's it! No build process or server needed.

## How to Use

- Move your mouse over the grid cells to draw (cells turn black)
- Click the "Clear" button to reset the canvas

## Tech Stack

- HTML5
- CSS3 (Flexbox)
- Vanilla JavaScript
